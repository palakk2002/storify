# 🎂 Birthday Special for Ankit Sir (Cutu)

This is a highly interactive, aesthetic birthday wish web experience.

## ✨ Features
- **Landing Screen**: Animated heading with floating particles.
- **Interactive Hub**: 4 clickable sections (Surprise, Special, Memory Lane, Secret Message).
- **Animations**: Typing effects, card flips, blurred secret reveal, and horizontal timeline.
- **Celebration**: Confetti bursts and a grand finale.
- **Music**: Background music toggle (current music is a placeholder).

## 🚀 How to Run
1. Open the `birthday-special` folder.
2. Double-click `index.html` to open it in your browser.
3. Click "Play Music" at the top right for the full experience.

## 📸 How to Add Real Memories
To add your own memories and inside jokes:
1. Open `script.js`.
2. Find the `sections.memory` object.
3. Update the `memories` array with your own titles (`t`) and descriptions (`d`).
4. You can also update the `reasons` array in `sections.special`.

## 🎵 Changing the Music
To change the background music:
1. Find a music file you like (MP3).
2. Save it in the `birthday-special` folder.
3. Open `index.html`.
4. Update the `<source src="...">` tag with your filename.
